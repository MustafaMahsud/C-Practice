# Largest of Three Numbers

## Objective

Write a C program that accepts three numbers from the user and determines the largest among them using conditional statements.

## Concepts Practiced

* User input (`scanf`)
* Conditional statements (`if`, `else if`, `else`)
* Comparison operators
* Basic program logic

## Example

**Input**


Enter three numbers: 10 25 18


**Output**


The largest number is: 25


## How to Compile

```bash
gcc main.c -o Largest_Number
```

## How to Run

### Windows

```bash
Largest_Number.exe
```

### Linux/macOS

```bash
./Largest_Number
```
